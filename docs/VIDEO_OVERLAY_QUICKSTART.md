# GrapplingOverlay Video Overlay Quickstart

1. Download the GrapplingOverlay zip and unzip it to a folder you can find (like Desktop).
2. Double-click `GrapplingOverlay.exe` to launch the app.
3. Click **Open Video File (Overlay)**.
4. Pick an MP4 video file when the file picker opens.
5. Wait for the preview window to show the skeleton overlay; when it finishes, find the output at `outputs/pose_tracks.json` inside the same folder.
